# Our Future Wallet

SPA financiera para pareja hecha con HTML, CSS y JavaScript vanilla.

## Archivos principales

- `index.html`: estructura de la aplicacion.
- `estilos.css`: estilos, temas y responsive.
- `script.js`: estado local, formularios, dashboard, exportacion y persistencia.
- `assets/`: imagenes usadas por los perfiles.

## Uso local

Abre `index.html` directamente en el navegador o sirve la carpeta con:

```bash
python -m http.server 4173
```
